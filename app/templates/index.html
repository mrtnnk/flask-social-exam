Welcome to the Flask-Social example website.

This site aims to show the basic functionality provided by the Flask-Social extension for Flask, a Python web framework. The source code for this application can be found on github.

Currently this site allows users to register and subsequently establish connections with their Twitter and/or Facebook accounts. Upon establishing a connection users can then login without having to provide their password.
